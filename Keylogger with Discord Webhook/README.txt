PowerShell Keylogger for Discord Webhook
Introduction

This PowerShell script, developed by CaliNux, is a simple keylogger designed to log keystrokes and send them to a Discord webhook. Please note that using this script for any malicious purposes is strictly prohibited and may violate privacy laws.
